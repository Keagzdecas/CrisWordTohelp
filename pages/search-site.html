Search Site

View Site Screen

On this page, you will be able to View site information on the system.

Navigation process

The user clicks on the View all site tab on the side navigation bar. The system will display the view all sites screen.

Control Name

Functionality Description

This field is where the user would input their search details. Which the system will use that input to search for the site.

When clicked the system will perform the search function.

Site details Table

This table displays all the information with regards to the site.

5

Delete Button

This button deletes the selected site. This is delete function is restricted if the site in register or in use by a project or company in the system.

11

Submit Button

This button will be enabled when the user has completed all required fields. This will then submit the sales representative personal modified information and redirect them back to the sales representative home screen.
