View Tickets

View Tickets Assigned

On this page, as a Technician you will be able to view the tickets that are assigned to you, allowing you to perform actions on tickets.

Navigation process

The preloaded Technician needs to login into the system. Once they login, they will be directed to the Home page.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Download Technical Site Assessment

When clicking this button, your browser will begin a download of the Technical Site Assessment that was uploaded to the ticket.

2

Select File from PC button

When clicking this button, your file explorer on your PC will open, allowing you to select your Technical Site Assessment to the system.

3

Upload Technical Site Assessment to ticket

When clicking this button, the previously selected file from the system will be uploaded onto the system.

4

Change Ticket Status to ‘In Progress’ Radio Button

When clicking this button, the Status of the ticket will be changed to ‘In Progress’.

5

Change Ticket Status to ‘Complete’ Radio Button

When clicking this button, the Status of the ticket will be changed to ‘Complete’.
