Generate Ticket

Generate Ticket

On this page, you will be able to generate a ticket that will be linked to a technician.

Navigation process

The preloaded Sales Rep needs to login into the system. Once they login, they will be directed to the Sales Home page. Here they will click on the ‘Tickets’ Button as seen below.

 

Once the view tickets page has loaded the generate ticket button is clicked as seen below.

 

 

 

 

 

 

 

 

 

 

 

 

 

Layout of the Page

 

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Label to display Request for quote

This label displays the relevant request for quote.

2

Label to display quote items

This label displays the relevant quote items that are linked to the project.

3

Search employee button

When clicking this button, the system will redirect to the Search employee screen

4

Description textbox

Sales reps should enter the necessary ticket details into this textbox.

5

Deadline datepicker

Sales reps should select the deadline for the ticket they are creating.

6

Back Button

When clicking this button, the system will redirect you to the view tickets screen.

7

Submit button

The submit button will display once all necessary inputs including selecting a employee has been done. When selecting this button, a new ticket will be saved to the system and the selected technician will be emailed.

 

 

 

 

 

 

Search Employee page

 

 

Reference Number

Control Name

Functionality Description

1

Search textbox

The sales rep can input search details here to find specific employees.

2

Profile Picture

This is where the employees profile picture will display. If there is a problem with the profile picture it will display as above.

3

Labels

These labels display all the relevant details for a employee including email, cell phone and tickets assigned.

4

Assign button

When selecting this button, a employee is selected for the specific ticket and you will be redirected to the generate ticket screen.
