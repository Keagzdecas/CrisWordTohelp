Search Labour Types

Search Labour Type

On this page, the stock user will be able to navigate and search for a Labour Type on the system.

Navigation process

The preloaded Stock user needs to login into the system. Once they login, they will be directed to the products page. Here they will click on the ‘Labour Type’ Button as seen below.

 

 

 

 

 

 

 

 

 

 

 

 

 

Then the Labour Type page loads, the stock user will input the name of the existing labour type that they are looking for and proceed to select the search button.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

Labour Type Search Textbox

Stock User should input the Labour Type name that they intend to search into this Textbox

Search Button

When the Stock user clicks this button, the Labour Type name that matches the input data will load on to the screen
