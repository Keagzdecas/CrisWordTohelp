Admin View Meetings

Admin Meetings

On this page, you will be able to view your schedule of meetings, as well as accept or decline the meetings.

Navigation process

The Admin needs to login into the system. Once they login, they will be select the meetings option on the sidebar in the Admin Tools Page. Thereafter, they will be directed to the Admin Meetings page.

 

 

 

 

 

 

 

 

 

 

 

Layout of the Page

On the click of a meeting, a pop-up will appear that will allow you to accept or decline the meeting.

 

 

 

 

 

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Calendar Today Button

When clicking this button, the calendar will display the current day and the schedule for the day.

2

Calendar day Button

When clicking this button, the calendar will display the selected day and the schedule for that day.

3

Calendar Work Week Button

When clicking this button, the calendar will display the selected work week and the schedule for that week.

4

Calendar Month Button

When clicking this button, the calendar will display the selected month and the schedule for that month.

5

Meeting

When clicking this meeting, the system will respond by providing a popup with more detailed information about the selected meeting.

6

Accept Meeting Button

When clicking this button, the Admin will confirm their attendance and accept the meeting.

7

Decline Meeting Button

When clicking this button, the Admin will deny their attendance and decline the meeting.

8

Back Button

This will close the meeting and navigate the Admin back to the calendar.
