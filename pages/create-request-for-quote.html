Create Request for Quote

Create request for quote

On this page, you will be able to create a request for quote on behalf of a client.

Navigation process

The preloaded Sales Rep needs to login into the system. Once they login, they will be directed to the Sales Home page. Here they will click on the ‘Request for quote’ Button on the sidenav as seen below.

Once the view request screen has loaded the create request for quote button is clicked as seen below.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Label to display Client Name

This label displays the selected client to create the request for quote for.

2

Search client button

When clicking this button, the system will redirect to the Search client screen

3

Request for quote textbox

Sales reps should enter the necessary request for quote details into this textbox.

4

Back Button

When clicking this button, the system will redirect you to the sales rep home screen.

5

Submit button

The submit button will display once all necessary inputs including selecting a client and entering a request for quote has been done. When selecting this button, a new request for quote will be saved to the system.

These labels display all the relevant details for a client including Date of birth, id number, gender, email and cell phone number.

When selecting this button, a client is selected for the specific request for quote and you will be redirected to the create request for quote screen.
