Modify Vehicle Type

Modify Vehicle Type

On this page, you will be able to Modify the Vehicle Types on the system.

Navigation process

The Admin will need to login into the system. Once they login, they will need to click on the Vehicle Management Tab in the Nav Bar. A new side menu will load. The admin must then press the Vehicle Type Button. The system will then direct the Admin to the Modify Vehicle Type Page.

Functionality Description

Vehicle Type Textbox

If the Vehicle Type name has changed, the Admin should input the new Vehicle Type name.

2

Back Button

When clicking this button, the system will redirect the Admin back to the View Vehicle Type Page and the modification of the Vehicle Type has been Cancelled.

3

Update Button

Clicking this button will save the changes made to the Vehicle Type name to the Database.
