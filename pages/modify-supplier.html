Modify Supplier

Modify Supplier

On this page, you will be modifying a supplier in the system.

Navigation process

 

The preloaded Stock User needs to login into the system. Once they login, they will be directed to the navigation bar. Here they will click on the ‘Supplier’, followed by the ‘Modify Supplier’ Button as seen below.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Supplier Name Input Field

The stock user enters the updated supplier name of the supplier.

2

City Input Field

The stock user enters the updated city of the supplier.

3

Postal Code Input Field

The stock user enters the updated postal code of the supplier.

4

Fax Number Input Field

The stock user enters the updated fax number of the supplier.

5

Contact Number Input Field

The stock user enters the updated contact number of the supplier.

6

Province Input Field

The stock user enters the updated province of the supplier.

7

Address Input Field

The stock user enters the updated address of the supplier.

8

Modify Button

When clicking this button, the system will you update the supplier in the system with the information given in the input fields above.

9

Cancel Button

When clicking this button, the system will respond by cancelling the request to modify a supplier and navigate you back to the supplier page.
