Search Products

Search Product

On this page, the stock user will be able to navigate to and Sign-out a product from the system.

Navigation process

The preloaded Stock user needs to login into the system. Once they login, they will be directed to the products page. They will be able to search from this page.

 

 

 

 

Page Layout

 

Page Detailed Description of the page layout

 

Reference Number

Control Name

Functionality Description

1

Product Search Input Field.

Stock User should input the product name of the product they want to search into this Input Field.

2

Search Button

When the Stock user clicks this button, the product will be searched for up and displaced when located.

 

 

Delete product

On this page, the stock user will be able to navigate to and delete a product from the system.

Navigation process

 

The preloaded Stock user needs to login into the system. Once they login, they will be directed to the products page. They will be able to delete from this page. If the product is not ordered then this will proceed without any problems.
