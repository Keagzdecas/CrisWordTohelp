Search and Delete Vehicle

Search and Delete Vehicle

On this page, you will be able to on the system.

Navigation process

The Admin will need to login into the system. Once they login, they will need to click on the Vehicle Management Tab in the Nav Bar. A new side menu will load. The admin must then press the Vehicle Button in the sidenav.

 

 

 

Layout of the Page

 

 

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Search Text box

The Admin inputs the License Number of the vehicle to search for the vehicle.

2

Search Button

When this button is clicked, the system will search for the vehicle based on the license number in the search textbox.

3

Add Vehicle Button

Clicking this button will direct the Admin to the Add Vehicle Page

4

Delete Button

Clicking this button will delete that specific vehicle from the system.

5

Modify Button

Clicking this button will direct the User to the Modify Vehicle Page where they will be able to Modify the Vehicles details

6

Vehicle Table View

This Table View contains the details of the vehicle.
