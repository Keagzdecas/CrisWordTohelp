Generate Project

Generate Project

On this page, you will be able to generate a project for the client.

Navigation process

The preloaded Sales Rep needs to login into the system. Once they login, they will be directed to the Sales Home page. Here they will click on the ‘Request for quote’ Button on the sidenav as seen below.

 

 

 

Once the view request screen has loaded the view quotes button is clicked as seen below.

Once the view quote screen is loaded the generate quote button can be clicked if a project has not been generated and the quote is accepted as seen below.

 

 

 

 

 

Layout of the Page

 

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Client label

This label displays the client a project is being created for.

2

File upload

When clicking this button, the system will open a file selector where the sales rep will select a relevant profile picture for the project. 

3

Name Textbox

This is where a sales rep will input the project name.

4

End date datepicker

This is where a sales rep will select the chosen end date for a project.

5

Back button

When clicking this button, the system will redirect you to the view quote screen.

6

Submit button

When selecting this button, a project will be saved on the system and the system will redirect you to the home screen.
