Add Supplier

Add Supplier

On this page, you will be adding new suppliers to the system.

Navigation process

 

The preloaded Stock User needs to login into the system. Once they login, they will be directed to the navigation bar. Here they will click on the ‘Supplier’, followed by the ‘Add Supplier’ Button as seen below.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Supplier Name Input Field

The stock user enters the supplier name of the supplier.

2

Contact Number Input Field

The stock user enters the contact number of the supplier.

3

City Input Field

The stock user enters the city of the supplier.

4

Province Input Field

The stock user enters the province of the supplier.

5

Postal Code Input Field

The stock user enters the postal code of the supplier.

6

Address Input Field

The stock user enters the address of the supplier.

7

Fax Number Input Field

The stock user enters the fax number of the supplier.

8

Add Button

When clicking this button, the system will you create a new supplier in the system with the information given in the input fields above.

9

Cancel Button

When clicking this button, the system will respond by cancelling the request to add a new supplier and navigate you back to the supplier page.
