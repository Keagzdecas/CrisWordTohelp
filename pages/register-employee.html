Register Employee

Register Employee

On this page, you will be able to register employees onto the system

Navigation process

The preloaded Administrator need to login into the system. Once they login, they will be directed to the Admin Tools page. Here they will click on the ‘Add’ Button as seen below.

 

 

Layout of the Page

 

 

 

 

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Upload an image from PC button

When clicking this button, your file explorer on your PC will open, allowing you to upload your profile picture to the system.

2

Upload a Webcam Image Button

When clicking this button, a webcam view will be displayed as well as three new buttons namely “Take A Picture Button, Take Another Button, Save Image Button”.

3

Take A Picture Button

When clicking this button, the webcam will capture and display the current image in the Webcam Image View.

4

Take Another Button

When clicking this button, the Webcam Image View will clear, allowing you to take another picture.

5

Save Image Button

When clicking this button, the system will save the current image in the Webcam Image View as the new stock users Profile Picture.

6

Webcam Image View

This Webcam Image View will display the Webcam current image as well as the Image the webcam has captured.

7

Employee Name Textbox

Admins should input the stock users Name into this Textbox

8

Employee Surname Textbox

Admins should input the stock users Surname into this Textbox

9

Employee Email Textbox

Admins should input the stock users Email into this Textbox

10

Employee ID Number Textbox

Admins should input the stock users ID Number into this Textbox

11

Employee Contact Number Textbox

Admins should input the stock users Contact Number into this Textbox

12

Employee Username Textbox

Admins should input the stock users Username into this Textbox

13

Employee Password Textbox

Admins should input the stock users Password into this Textbox

14

Employee Gender Dropdown Menu

When clicking on the dropdown menu, a list of Genders i.e. Male, Female, and other will be displayed. The Admin should select which option applies to the stock users.

15

Employee Type Dropdown Menu

When clicking on the dropdown menu, a list of Employee Types i.e. Admin, Technician, Stock User and Sales Representative will be displayed. The Admin should select stock users’ option.

16

Employee User Role ID Dropdown Menu

When clicking on the dropdown menu, a list of User Roles i.e. Admin, Technician, Stock User, Client and Sales Representative will be displayed. The Admin should select the stock user option.

17

Employee Date of Birth Date Picker

When clicking on this date picker, the admin will be presented with a date picker. They should use this date picker to select the stock users date of birth.

18

Cancel Button

When clicking this button, the Admin will be redirected to the Admin Tools page and the Register the stock users has been cancelled

19

Submit Button

When all the fields in the Register Employee form have filled, this button will become clickable. When the Admin clicks this button, the new stock user will be saved to the database and can now start using the system.
