Modify Technician

Modify Technician

On this page, you will be able to modify other registered admins on the system.

Navigation process

The Admin will need to login into the system. Once they login, they will need to click on the User Management Tab in the Nav Bar. A new side menu will load. The admin must then press the Technician Button. Once the page has loaded, the admin should then press on the modify button on the Technician they want to modify.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Upload an image from PC button

When clicking this button, your file explorer on your PC will open, allowing you to upload your profile picture to the system.

2

Upload a Webcam Image Button

When clicking this button, a webcam view will be displayed as well as three new buttons namely “Take A Picture Button, Take Another Button, Save Image Button”.

3

Take A Picture Button

When clicking this button, the webcam will capture and display the current image in the Webcam Image View.

4

Take Another Button

When clicking this button, the Webcam Image View will clear, allowing you to take another picture.

5

Save Image Button

When clicking this button, the system will save the current image in the Webcam Image View as the new Technicians Profile Picture.

6

Webcam Image View

This Webcam Image View will display the Webcam current image as well as the Image the webcam has captured.

7

Employee Name Textbox

Admins should input the Technicians Name into this Textbox if it changed.

8

Employee Surname Textbox

Admins should input the Technicians Surname into this Textbox if its changed.

9

Employee Email Textbox

Admins should input the Technicians Email into this Textbox if its changed.

10

Employee Contact Number Textbox

Admins should input the Technicians Contact Number into this Textbox if its changed.

11

Employee Gender Dropdown Menu

When clicking on the dropdown menu, a list of Genders i.e. Male, Female, and other will be displayed. The Admin should select which option applies to the Technician.

12

Employee Date of Birth Date Picker

When clicking on this date picker, the admin will be presented with a date picker. They should use this date picker to select the Technicians Date Of Birth.

13

Cancel Button

When clicking this button, the Admin will be redirected to the Search Technician page and the Modify Technician has been cancelled

14

Submit Button

When the Admin clicks this button, the new details of the Technician will be saved to the database.
