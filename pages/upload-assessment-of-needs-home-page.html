Upload Assessment of Needs Home Page

Upload Assessment of Needs Screen

On this page, you will be able to upload an assessment of needs file to the system.

Navigation process

The user clicks on the Homepage tab on the side navigation bar which will then redirect them to the sales representative Home page screen. The user then clicks on the Assessment Button in the project they wish to upload to. This will redirect them to the upload assessment of needs screen. The user then clicks on the choose file button which will open file explore on their computer. They select the file they wish to upload and click open button on file explore. This file will load into the system, the user then clicks on the upload button and they are redirect back to the sales representative’s home screen.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Upload Assessment of needs Card

This card displays the upload of assessment needs function.

2

Choose file button

When clicked this button opens file explore on your computer and allows the user to select a file to be uploaded. The user must select a file and click open on file explore. This file is then uploaded into the system if submit button is clicked.

3

Back Button

This button cancels the upload of assessments of needs and redirects the user back to the sales representative home back.

4

Submit Button

This submits the form and uploads the selected file to the system if a file is selected.
