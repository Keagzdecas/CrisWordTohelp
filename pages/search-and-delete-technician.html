Search and Delete Technician

Search and delete Technician

On this page, you will be able to Search and Delete a Technician.

Navigation process

The Admin will need to login into the system. Once they login, they will need to click on the User Management Tab in the Nav Bar. A new side menu will load. The admin must then press the TechnicianButton.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Search Textbox

To search for a Technician, insert the name or surname of the Technician into the textbox.

2

Search Button

When clicking this button, the system will search and display the Technician found in the system based on the name or surname in the search textbox.

3

Modify Button

When clicking this button, the system will direct the Admin to the Modify Technician page where the Admin can modify the details of that specific Technician

4

Delete Button

When clicking this button, the system will delete this specific Technician from the system.

5

Table View

This Table contains the details of the Technician
