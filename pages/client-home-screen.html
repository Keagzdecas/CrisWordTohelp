Client Home Screen

Client Home Screen

On this page, you will be able to view the clients home page with projects and complaints being displayed from the system.

Navigation process

The Client needs to login into the system. Once they login, they are redirected to the Client Home screen where any projects or complete details are displayed for their viewing.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Modify Client Button

This Displays the client name and surname and is clickable. When click is redirects the user to the modify client screen where they can modify their profile and reset their password.

2

Logout Button

This button prompts the logout dialog. This then logs the client out the system and redirects them back to the Website Home page.

3

Home Button

This button redirects the user back to their home page.

4

Create Request for Quote button

This button displays the create request for quote screen where the client can create a request for quote.

5

View Request for Quote button

This button when clicked displays the view request for quote screen where the client can view the past request for quote, they have placed.

6

Meetings Button

This button displays the view meeting screen which contains a calendar view where the client can view meeting scheduled.

7

Quotes Button

This redirects the user to the View Quotes screen where they can view quotes.

8

Project Button

This button displays the view project screen where the client and view their projects.

9

Complaint Feedback

This button displays the view complaint screen where the client can view their complaints that they have previously lodged.

10

Project Details Table

This Table displays the details of the client’s current projects

11

This Table Displays the Complaint Table

This table displays the complaint information that a client has previous placed.
