Modify Sales Rep Profile

Modify Sales Representative Screen

On this page, you will be able to modify a sales representative information on the system.

Navigation process

The user clicks on their name on the top navigation bar which will then redirect them to the modify sales representative screen where their personal information will be prepopulated. They then enter the fields they wish to edit and click on the submit button.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Layout of Page (Modify Sales Representative)

 

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Modify Sales Representative card

This card displays the information form with prepopulated fields with the sales representative’s personal information to be edited.

2

Sales Representative Profile Picture

This is an image of the sales representative profile image.

3

Upload An image from PC

This button unhides a button that allows the user to click and opens file explore on their computer. The user will then select a File and click “open” on file explore which they will then be redirected back to the modify Page with their file Uploaded.

4

Upload a Webcam image

This button unhides the webcam function which with 3 other buttons. These allow for a webcam image to be taken and captured as the user’s profile picture.

5

Name input Field

This field is where the user must insert their personal name.

6

Surname input Field

This field is where the user must insert their personal Surname.

7

Email input field

This field is where the user must insert their personal email address.

8

Contact input field

This field is where the user must insert their personal contact number.

9

Gender dropdown menu

This is a dropdown menu where the user must select their gender.

10

Date of Birthday Dropdown menu

This is a dropdown menu where the user must select their personal date of birth.

11

Submit Button

This button will be enabled when the user has completed all required fields. This will then submit the sales representative personal modified information and redirect them back to the sales representative home screen.
