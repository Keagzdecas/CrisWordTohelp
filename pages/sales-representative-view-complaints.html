Sales Representative View Complaints

View Complaints Screen

On this page, you will be able view Complaints on the system.

Navigation process

The user clicks on the Homepage tab on the side navigation bar which will then redirect them to the sales representative Home page screen. The user then clicks on the View Complaint Button on the sales representative side navigation bar. This will redirect them to the View Complaints screen.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

View Complaint Table

This table displays all the complaint details for the sales representative to view.

2

Complaint Status Description

This displays the description of the current complaint status.

3

Date Description

This description displays the date in which the complaint was lodged by the client.

4

Client Complaint Description.

This description displays the complaint that the client lodge that the sales representative needs to respond too.

5

Due in description

This Description displays the days till a response from the sales representative is due. This status changes colour according to the closeness of the date.

6

Response to Complaint Button

This button redirects the user to the response to complaint screen where the sales representative is able to reply to the clients complaint. An email is created and sent to the client with the sales representative reply.
