Modify Product Type

Modify product Type

On this page, the stock user will be able to navigate and modify a product Type on the system.

 

 

 

Navigation process

The preloaded Stock user needs to login into the system. Once they login, they will be directed to the products page. Here they will click on the ‘Product Type’ Button as seen below.

 

 

 

 

Then The stock user will select the edit button of an existing product type.

 

 

Layout of the Page

 

 

 

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Product Type Description Textbox

Stock User should input the Product Type name into this Textbox

2

Cancel Button

When clicking this button, the Stock user will be redirected to the Products page and the Register Product Type has been cancelled

3

Submit Button

When the Stock user clicks this button, the modified product Type will be saved to the database.
