View User Roles

User Role

On this page, you will be able to view the user roles in the system, from this page you will be able to search for specific user roles, delete user roles, as well as navigate to creating new ones.

Navigation process

The preloaded Administrator need to login into the system. Once they login, they will be directed to the Admin Tools page. Here they will click on the ‘View User Role’ Button as seen below.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Search Input

When the admin types in the input field, the system will respond by displaying only the user roles that match the search criteria.

2

Create New User Role Button

When clicking this button, the system will navigate you to a page that allows you to add a new user role to the system.

3

Delete User Role Button

When clicking this button, the system will delete the user role from the system; provided the user role is not linked to any users.

 

 

Delete User Role

On this page, you will be able to delete a user role in the system.

Navigation process

The preloaded Administrator need to login into the system. Once they login, they will be directed to the Admin Tools page. Here they will click on the ‘View User Role’ Button, from where they can click ‘Delete User Role’ Button from the landing page, as seen below.
