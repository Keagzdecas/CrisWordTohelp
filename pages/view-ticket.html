View Ticket

View Tickets Screen

On this page, you will be able view tickets on the system.

Navigation process

The user clicks on the Homepage tab on the side navigation bar which will then redirect them to the sales representative Home page screen. The user then clicks on the Tickets Button in the project they wish to View tickets for. This will redirect them to the View Tickets screen.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Project Name Heading

This Heading Displays the project name associated with the tickets.

2

Generate Ticket button

When clicked this button redirects the user to the generate ticket screen where they can create a new ticket and assign it to a technician.

3

Back Button

This button redirects the user back to the sales representative home page.

4

Search Field

This search field is used to search tickets by description. This will filter the tickets and display according to the searched input.

5

Search Button

This confirms the search and runs the search function.

6

Ticket Card

This card displays the information of the ticket.

7

Edit Button

This button allows the user to edit a create ticket.When click it redirects the user to the update ticket page.

8

Delete Button

This button displays the confirm delete prompt and if confirmed it deletes the ticket.
