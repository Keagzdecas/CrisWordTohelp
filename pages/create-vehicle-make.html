Create Vehicle Make

Create Vehicle Make

On this page, you will be able to modify Vehicle Makes on the system.

Navigation process

The Admin will need to login into the system. Once they login, they will need to click on the Vehicle Management Tab in the Nav Bar. A new side menu will load. The admin must then press the Vehicle Make Button. Once the page loads, they will need to click on the Add Vehicle Make button.

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

Vehicle Make Description Textbox

If the Vehicle Type name has changed, the Admin should input the new Vehicle Type name.

2

Back Button

When clicking this button, the system will redirect the Admin back to the View Vehicle Type Page and the modification of the Vehicle Type has been Cancelled.

3

Submit Button

Clicking this button will save the new Vehicle Make to the Database. This button wont be clickable if nothing is entered.
