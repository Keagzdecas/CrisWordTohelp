View Request For Quote

View Request for Quotes Screen

On this page, you will be able to view their past request for quotes on the system.

Navigation process

The Client needs to login into the system. Once they login, they are redirected to the Client Home screen where any projects or complete details are displayed for their viewing. They then click on the View request for quote tab on the side navigation bar which will open the View Request for Quote Screen.

 

 

 

 

 

 

Layout of Page (View Request for Quote Screen)

 

Detailed Description of the page

Reference Number

Control Name

Functionality Description

1

View Request for Quote Details Table

This table displays details with regards to the past request for quotes that the client has placed.

2

Request for Quote Status Display

This displays the current status of the request for Quote. The colour displayed depends on the status. Green is when the quote has been created. Orange is when a quote is pending.

3

Request for Quote Description

This displays the client’s description of what they have requested to be quoted for.

4

Sales Representative Description

This displays the sales representative’s addition or edit to the quote.
